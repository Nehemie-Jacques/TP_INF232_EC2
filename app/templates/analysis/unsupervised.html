À propos
{% if method == 'kmeans' %}

K-Means : partitionne les données en k groupes en minimisant l'inertie intra-cluster.

Score silhouette : proche de 1 = bons clusters. Proche de 0 = clusters qui se chevauchent.

Méthode du coude : cherchez le "coude" dans la courbe d'inertie.

{% elif method == 'dbscan' %}

DBSCAN : regroupe les points denses, identifie les outliers comme "bruit".

Avantage : découvre des clusters de forme arbitraire, pas besoin de spécifier k.

Inconvénient : sensible aux paramètres ε et MinPts.

{% else %}

CAH : construit une hiérarchie de clusters, visualisée par le dendrogramme.

Ward : minimise la variance totale — méthode la plus robuste en général.

Dendrogramme : la hauteur de coupure détermine le nombre de clusters.

{% endif %}
